# Security Audits

Security compliance and auditing for Windows Server environments.

## Structure

```
audits/
├── cis-benchmarks/       # CIS Windows Server compliance scanning
├── security-baselines/   # Microsoft Security Compliance Toolkit
└── compliance-reports/   # Automated compliance reporting
```

## Focus Areas

### CIS Benchmarks
- Windows Server 2022 CIS Level 1 & 2
- Automated compliance scanning scripts
- Remediation scripts for common findings
- PowerShell 5.1 compatible

### Security Baselines
- Microsoft Security Compliance Toolkit integration
- Group Policy baseline exports
- Configuration assessment scripts

### Compliance Reports
- Automated report generation
- Dashboard scripts for compliance tracking
- Export to HTML/CSV/JSON formats

## Requirements

- PowerShell 5.1+
- Administrator privileges
- Windows Server 2022

## Usage

Scripts in this directory should be tested in a lab environment before production use.
